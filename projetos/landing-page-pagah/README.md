# Landing Page Pagah — 4 versões comparativas

4 versões da landing page da Pagah construídas pra comparação e escolha. Mesma copy, 4 moods visuais.

| Rota | Versão | Mood |
|---|---|---|
| `/v1-premium-dark` | A — Premium Fintech Dark | Stripe-meets-Linear, dark + halo amarelo |
| `/v2-editorial` | B — Editorial Bold | Tipografia gigante, mood revista |
| `/v3-living-tech` | C — Living Tech / Bento | Bento grid com perpetual motion |
| `/v4-heritage` | D — Pagah Heritage Refined | Identidade Pagah elevada |
| `/` | Switcher | Cards-preview pras 4 |

## Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4 (CSS-first config)
- Framer Motion
- TypeScript strict

## Dev

```bash
npm install
npm run dev
```

Abre em http://localhost:3000

## Deploy

`vercel.json` já configurado. `vercel --prod` quando quiser subir.
